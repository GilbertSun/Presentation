2. 如何拆分组件

原子级别的组件

  • 自己创造一部分
  • 第三方改造一部分

页面或场景级别的组件

这种级别的组件, 目的不是复用而是切割页面的逻辑, 用原子级别的组件和自己实际的页面逻辑进行堆砌

3. 如何封装组件

声明式的组件

市面上声明式的组件库是很少的, 有也被限制在了某个平台

{ant-design, bootstart-react, ...} => React

{ant-design-vue, vuestarp, ...} => Vue

{Angular-bootstarp, ...} => Angular

自己封装时, 遵循 Vue 的原则

命令式的组件

以声明式的标签形式使用, 内部使用 vm.ready 进行命令式的初始化
这种形式能让我们应用上大部分的命令式组件库
